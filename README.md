# picamera-stream
Flask server that hosts a real time stream from pi camera.

## what it does
A flask server 

## how to run
1. Open terminal
2. ```cd PATH\TO\FOLDER```
3. Activate the virtual environment ```source /pienv/source/activate```
4. Run ```python3/python serve.py```
5. Navigate to ```http://ip_to_server/camera```
